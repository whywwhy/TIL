# file-tree-generator

### 1️⃣ file-tree-generator?
<hr/>

꽤 신기한 확장자가 있어서 가져와 봤다

여태껏 README.md 파일에 파일 구조를 옮겨 적을 때 <br/>
터미널에서 tree 명령어를 쳐서 나온 결과물을 그대로 복붙해야하는데

노션에 적을 때 너무 불편해서 찾아보다 발견하게 된 vscode extension 이다.
